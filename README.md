# Candy Crush

This repository contains two versions of the same game:

| Folder | Language | Description |
|--------|----------|-------------|
| [`Candy-Crush-Cpp/`](Candy-Crush-Cpp/) | C++ | Original console game (unchanged) |
| [`Unity-CandyCrush/`](Unity-CandyCrush/) | C# / Unity | Advanced remake with animations, special candies, and UI |

## Original C++ Version

A simple Candy Crush clone for the Windows console, built as a Programming Fundamentals project at NUCES-FAST, FSD.

**Requirements:** C++ compiler (Visual Studio recommended)

Compile and run `Candy-Crush-Cpp/Source.cpp`.

## Unity Version

See [`Unity-CandyCrush/README.md`](Unity-CandyCrush/README.md) for setup instructions.

Quick start: open `Unity-CandyCrush` in Unity Hub, then use **Candy Crush => Setup Full Project** from the menu.

## Improvements Welcome

Pull requests and improvements are welcome for either version.
